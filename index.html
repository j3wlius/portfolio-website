What I Do

Responsive Design

I employ flexible layouts and techniques like media queries for adaptability & make sure websites look and function flawlessly on different devices, from desktops to smartphones, ensuring a seamless user experience.

Accessibility Focus

Implement accessibility guidelines (WCAG) throughout development & prioritize designing websites that everyone can use, regardless of ability, by considering keyboard navigation, color contrast, and semantic structure.

UI Development

I translate design mockups into functional front-end code using HTML for content, CSS for styling, and JavaScript to add interactive elements.

Collaboration & Growth

I contribute effectively to team projects, evolve professionally & work closely with designers and back-end developers for seamless integration, and stay updated on the latest web development technologies and trends.
